🪟 Active Directory Security Portfolio

Overview

This repository contains hands-on Active Directory security labs focused on Windows enterprise environments, identity management, access control, privilege security, and system hardening.

The projects demonstrate practical knowledge of securing Microsoft Windows infrastructure and implementing defensive security controls.
